# Pepe Website Setup

This project has been set up with Mantine UI and React Router to create a modern portfolio website.

## Features Added

- **Mantine UI Components**: Modern, accessible UI components
- **React Router**: Client-side routing for navigation
- **Three Main Pages**:
  - **Projects**: Showcase of portfolio projects with cards, images, and technology badges
  - **Biography**: Personal information, skills, experience timeline, and education
  - **Contact**: Contact form and social media links

## Pages Overview

### 1. Projects Page (`/projects`)

- Grid layout with project cards
- Project images, descriptions, and technology badges
- Live demo and code repository links
- Responsive design

### 2. Biography Page (`/biography`)

- Personal profile with avatar
- Skills and technologies section
- Professional experience timeline
- Education timeline
- Two-column responsive layout

### 3. Contact Page (`/contact`)

- Contact form with validation
- Contact information cards
- Social media links
- Response time information

## Navigation

- Clean header navigation with active state indicators
- Mobile-responsive burger menu
- Smooth routing between pages

## Dependencies Installed

- `@mantine/core` - Core Mantine components
- `@mantine/hooks` - Mantine hooks for responsive design
- `@tabler/icons-react` - Icon library
- `react-router-dom` - Client-side routing

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173`

## Customization

- Update personal information in the Biography page
- Replace project data in the Projects page
- Modify contact information in the Contact page
- Customize colors and styling through Mantine's theme system

The website is fully responsive and follows modern web development best practices with clean, maintainable code structure.
